    节拍器（Beat）¶

    此库适用于需要精确节拍同步的应用场景，如音乐游戏、音频可视化或交互式媒体项目。

    主要模块与方法¶

    配置设置¶

    • SetBPM(new_bpm)：设置BPM值（需为正数）。
    • SetCycleLength(length)：设置节拍循环长度（如4拍一小节）。

    状态获取¶

    • GetBeat()：返回当前节拍的整数部分和小数部分。
    • GetBar()：返回当前小节号和节拍在循环中的位置。

    事件处理¶

    • RegisterEvent(event_type, handler)：注册事件回调（支持beat（节拍触发）和reset（重置）事件）。

    节拍控制¶

    • SetBeat(beat)：直接设置当前节拍（触发重置事件）。
    • Reset()：重置节拍到0（保留BPM设置）。

    状态更新¶

    • Update(dt)：基于时间增量（秒）更新节拍状态，自动触发节拍事件和循环检测。

    条件检测¶

    • OnBeat(target_beat)：检查是否到达目标节拍（支持循环内的相对节拍检测）。
    -- 导入节拍管理库
local Beat = require("Beat.lua")

-- 示例1：基础节拍跟踪与事件触发
function setupBasicBeatTracking()
    -- 设置BPM为120（常见流行音乐速度）
    Beat.SetBPM(120)

    -- 注册节拍事件：每整数拍触发
    Beat.RegisterEvent("beat", function(current_beat)
        print(string.format("第 %d 拍触发！", current_beat))
    end)

    -- 注册循环重置事件
    Beat.RegisterEvent("reset", function(beat)
        print(string.format("节拍重置至：%.2f", beat))
    end)
end

-- 示例2：游戏中的节拍同步逻辑
function gameBeatExample()
    Beat.SetBPM(140)  -- 设置较快节奏
    Beat.SetCycleLength(4)  -- 4拍一个循环

    -- 模拟游戏更新循环（dt为帧时间差）
    function gameUpdate(dt)
        Beat.Update(dt)  -- 更新节拍状态

        -- 检测特定节拍点（如每小节第1拍）
        if Beat.OnBeat(0) then
            spawnEnemy()  -- 在强拍生成敌人
        end

        -- 检测循环内的第3拍
        if Beat.OnBeat(2) then
            triggerSpecialEffect()  -- 触发特效
        end

        -- 实时获取节拍信息用于UI显示
        local integer_beat, fractional_beat = Beat.GetBeat()
        updateBeatUI(integer_beat, fractional_beat)
    end
end

-- 示例3：音乐可视化应用
function audioVisualizationExample()
    Beat.SetBPM(90)
    Beat.SetCycleLength(8)  -- 8拍一个乐句

    function onAudioFrame(dt)
        Beat.Update(dt)

        local current_bar, bar_beat = Beat.GetBar()

        -- 根据小节位置调整可视化强度
        if bar_beat < 2 then
            setVisualizerIntensity(0.5)  -- 弱拍阶段
        else
            setVisualizerIntensity(1.0)  -- 强拍阶段
        end

        -- 每小节第一拍突出显示
        if Beat.OnBeat(0) then
            triggerBarHighlight()
        end
    end
end

-- 示例4：手动节拍控制（用于调试或特殊场景）
function manualControlDemo()
    -- 直接跳转到特定节拍
    Beat.SetBeat(16.5)  -- 跳到第16.5拍

    -- 重置整个节拍系统
    Beat.Reset()
end

-- 运行示例
setupBasicBeatTracking()

-- 模拟时间流逝（每帧调用）
for i = 1, 100 do
    Beat.Update(1/60)  -- 假设60帧每秒
end
